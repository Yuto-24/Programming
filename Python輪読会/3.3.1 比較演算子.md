# 3.3.1 比較演算子

条件式を演算子を作るには，**比較演算子**(関係演算子とも)を用います．
C言語と同じなので，Cに馴染みがある人にとっては説明不要でしょう．

一覧は表1の通りです．

| 演算子 | 説明                   |
| ------ | ---------------------- |
| `==`   | 左辺と右辺は等しい     |
| `!=`   | 左辺と右辺は等しくない |
| `>`    | 左辺は右辺より大きい   |
| `<`    | 左辺は右辺より小さい   |
| `>=`   | 左辺は右辺**以上**     |
| `<=`   | 左辺は右辺**以下**     |
